# 원격 Repository clone 하기
- 원격 Repository에 push 된 프로젝트를 통째로 local disk로 복제하기
- 복제하여 저장할 부모 폴더에서 git bash 열기
- 원격 Repository에서 프로젝트 주소 복제해 오기
```
https://github.com/kimgukju/Biz-2023-04-505-Java.git
```
- 복제한 주소를 git bash 창에 붙여넣기
```bash
git clone https://github.com/kimgukju/Biz-2023-04-505-Java.git    newfolder
```
